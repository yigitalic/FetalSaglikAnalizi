# Fetal Sağlık Analizi

Bu proje, fetal sağlık verilerini analiz etmek ve görselleştirmek için geliştirilmiş Racket tabanlı bir uygulamadır.

## Dosyalar

*   **fetal_health.rkt**: Ana uygulama kodu. Veri analizi ve GUI işlemlerini içerir.
*   **generate_matrix.rkt**: Korelasyon matrisi oluşturmak için kullanılan yardımcı script.
*   **fetal_health.csv**: Analiz edilen veri seti.
*   **correlation_matrix.png**: Oluşturulan korelasyon matrisi görseli.

## Gereksinimler

*   Racket (DrRacket önerilir)
*   Gerekli Racket kütüphaneleri (örneğin `plot`, `math` vb.)

## Kurulum ve Çalıştırma

1.  Bu depoyu klonlayın:
    ```bash
    git clone https://github.com/yigitalic/FetalSaglikAnalizi.git
    ```
2.  `fetal_health.rkt` dosyasını DrRacket ile açın ve çalıştırın.
